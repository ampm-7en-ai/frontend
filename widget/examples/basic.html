Chat Widget Examples

1. Basic Floating Widget

The simplest way to add a chat widget to your website. This creates a floating button in the bottom-right corner.

<script src="../dist/chatbox-widget.js" data-agent-id="demo-agent-123" data-primary-color="#9b87f5" data-chatbot-name="Demo Assistant" data-welcome-message="Hello! I'm here to help you with any questions." data-button-text="Chat with us"></script>

2. Inline Chat Widget

Embed the chat widget directly into your page content.

<div id="inline-chat-container" style="width: 100%; height: 500px;"></div>

3. JavaScript API Control

Control the widget programmatically using JavaScript.

// Create widget programmatically const widgetId = window.ChatWidget.create({ agentId: 'demo-agent-123', primaryColor: '#9b87f5', chatbotName: 'JavaScript Assistant', mode: 'floating', suggestions: [ 'What can you help me with?', 'How do I get started?', 'Tell me about your features' ] }); // Update widget theme window.ChatWidget.update(widgetId, { primaryColor: '#ff6b6b', chatbotName: 'Updated Assistant' }); // Destroy widget window.ChatWidget.destroy(widgetId);

4. Advanced Configuration

Example with all configuration options.

const config = { agentId: 'demo-agent-123', primaryColor: '#9b87f5', secondaryColor: '#ffffff', fontFamily: 'Inter, sans-serif', chatbotName: 'Advanced Assistant', welcomeMessage: 'Welcome! I can help you with:', buttonText: 'Get Help', position: 'bottom-right', mode: 'floating', avatarUrl: 'https://example.com/avatar.png', suggestions: [ 'Product information', 'Technical support', 'Account questions', 'Billing inquiries' ], apiBaseUrl: 'https://api-staging.7en.ai' }; const widgetId = window.ChatWidget.create(config);
